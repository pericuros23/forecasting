SCO Forecasting Tool

A simple, browser-based forecasting tool for estimating SCO (Self-Checkout) install revenue across any retail client.
The app runs entirely in the browser and requires no login, setup, or downloads.

Built for quick scenario modeling, planning conversations, and revenue validation.

What This Tool Does

The SCO Forecasting Tool allows managers to:

Enter any client name

Define custom pricing for:

Full Function (FF) units

Narrow Core (NC) units

Model one or more SCO install events with:

Install date

Quantity per unit type

Forecast revenue through any future end date

Automatically calculate:

Active days per install (month-accurate, leap years included)

Monthly revenue totals

Per-install revenue breakdowns

All calculations follow the defined pricing inputs and apply consistent, transparent rules.

How to Use

Enter the client name

Input Full Function and Narrow Core pricing

Add one or more install events

Choose a forecast end date

Review:

Monthly revenue totals

Detailed per-install calculations

Export results if needed

Key Benefits

Client-agnostic — works for any retailer

Fast scenario testing — adjust pricing or quantities instantly

Accurate forecasting — month-level precision with leap-year support

No setup required — runs fully in the browser

Shareable — export results for review or planning

Intended Use

This tool is intended for:

Revenue planning

Scenario modeling

Install pacing discussions

Validation of expected SCO revenue impact

It is not a billing system and should be used as a forecasting and planning aid.
